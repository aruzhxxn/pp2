# Practice5 - Python RegEx

This project demonstrates the use of Python Regular Expressions (RegEx) to parse receipt data.

## Files

- raw.txt – contains the raw receipt text.
- receipt_parser.py – Python script that reads the receipt and extracts information using regular expressions.

## What the program does

The program reads the receipt text and extracts:

- product names
- prices
- date
- time
- payment method

## Technologies

- Python
- re (Regular Expressions module)

## How to run

1. Make sure both files are in the same folder.
2. Run the script:

python receipt_parser.py
